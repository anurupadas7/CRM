Please follow the following steps to run this CRMS project:

1. Download the github project in a zip file.

2. Extract the file and copy the "crm" folder to htdocs under xampp folder.

3. Open Xampp Control panel and run APache and MYSQL.

4. Open PHPMyAdmin in a browser tab, (http://localhost/phpmyadmin)

5. Create a database with name "crm_db".

6. Import the crm_db.sql file(given inside the zip package in SQL file folder under database) 

7. Run the script on a browser, http://localhost/crm to go to the customer dashboard.

8. You may register as a user following the given steps on clicking the Register button.

9. You may then log in as an administrator, at http://localhost/crm/admin/login.php using the following credentials:

    Username: admin
    Password: admin123

10. Check out the features and enjoy!
